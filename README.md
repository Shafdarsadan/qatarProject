# Qatar Company Website

A simple TypeScript-powered static website for a company providing services to people and businesses in Qatar.

## Available scripts

- `npm install` - install dependencies
- `npm run dev` - start development server
- `npm run build` - bundle the site for production
- `npm run preview` - preview the production build locally

## Project structure

- `index.html` - landing page
- `src/main.ts` - main entry point
- `src/style.css` - styles for the static website

## Notes

This is a starter site with a company overview, service cards, and contact section.
